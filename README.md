# FIM Referee Helper

## Bottom Line Up-Front
This project consists of a web application designed to assist FIRST in Michigan FIRST Robotics Competition Head Referees with their responsibilities.

## Roadmap
- [ ] User registration + profile service
- [ ] User event history
- [ ] Michigan event history
- [ ] Historical event referee rosters
- [ ] Opt-in referee directory (secured)
- [ ] Event request/election system
- [ ] Event assignment swaps
- [ ] Calendar
- [ ] Training Links
